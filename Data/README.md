## Dataset Information

This project uses a simulated hospitality dataset created for educational and portfolio purposes.

### Dataset Structure
- booking_date
- check_in_date
- property_id
- city
- room_class (Elite, Premium, Presidential, Standard)
- revenue_generated
- revenue_realized
- booking_status
- cancellation_flag

### Notes
- No real customer or company data is used.
- The dataset is designed to reflect realistic hotel booking behavior.
